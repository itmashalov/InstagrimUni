  • Education:
  • 1. Configuration and Pairing
    • 1.1 Education

      • These meters can be configured with EziView (for Windows) using a USB optical head. Some have test certificates, others have production certificates for pairing with an IHD/CAD. When EziView Is installed it will ask for the provided cd-key. When the key is entered it will generate another key which must be sent to EDMI team and they will send back the activation key.


    • 1.2 Work Experience

      • Once EziView is activated The proram will ask for username and password. The username is EDMI and the password is IMDEIMDE.After that you need to attach the optical head to the meter. Right click on Quick Connect and chose properties. Choose the right COM for this USB. After that click connect. This is only to test if the optical prob is connected. To be able to read and write to the meter you need to create a new station as a modem and set the details manualy.
